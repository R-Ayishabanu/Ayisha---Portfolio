# Ayisha Banu – Portfolio Website

This is a personal portfolio website built with HTML, CSS, and JavaScript. It is fully responsive, mobile-friendly, and styled with a modern gradient background. The project includes scroll animations, project showcases, and contact information.

## 🔗 Live Preview
You can deploy this website for free using GitHub Pages, Netlify, or Vercel.

## 🚀 Features
- Responsive design for all screen sizes
- ScrollReveal animations
- Gradient background theme
- Project showcase section
- Skills and About section
- Social media links
- Downloadable resume
- Contact form (via FormSubmit)

## 🛠️ Built With
- HTML5
- CSS3 (with Google Fonts and custom variables)
- JavaScript (Vanilla)
- ScrollReveal.js (for animation)
- RemixIcon (for icons)

1. Clone or download this repository.
